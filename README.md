# Personal Website

This is my personal website built from scratch using HTML, CSS, and JavaScript.

The website was created with a dark theme using black and blood-red colors to create a unique and modern design.

Live website:

https://zidi-site.netlify.app

## Features

* Responsive design for desktop and mobile devices
* Smooth scrolling navigation
* Mobile hamburger menu
* Spotify artist embeds
* Automatic GitHub project loading using GitHub API
* Social media section

## Technologies

* HTML5
* CSS3
* JavaScript
* GitHub API

## Project Structure

```
Personal-Website/

```

## How to run

1. Clone this repository:

```bash
git clone https://github.com/patriq128/Personal-Website.git
```

2. Open the project folder:

```bash
cd Personal-Website
```

3. Open `index.html` in your browser.

No additional setup is required.

## How it works

The website is built using basic frontend technologies without any frameworks.

The projects section uses the GitHub API to automatically fetch my repositories and display them on the website. Projects are filtered and sorted by the latest updates.

The navigation system, animations, and responsive behavior are handled using JavaScript and CSS.
